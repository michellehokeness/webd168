This is a wordle game that I created using HTML, CSS, and JavaScript. It is a game that allows you up to 5 guesses to figure out what a five letter random word is. If your word is the correct word, then the letters will be displayed in green. All incorrect letters guessed will be displayed in grey. If your word is not the correct word but contains some of the correct letters in the correct place, then the correct letters will be displayed in green. If your word is not the correct word but contains the correct letters in the wrong place, then the correct letters will be displayed in yellow. The game is a fun way to practice your spelling, vocabulary, memory, and critical thinking skills.
