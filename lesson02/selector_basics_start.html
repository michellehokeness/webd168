CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

Not Me!

My comfort level using CSS for styling webpages is fairly low. I am pretty good at using other resources to figure out how to code things. Some of those resources are: mdn web docs, W3Schools, and Google. I get a bit overwhelmed going to mdn web docs so I usually go to the other 2 listed first. Stack overflow can be helpful too. "CSS: The Missing Manual", is a nice resource, as well. I like how each section is marked by the green block on top of the page. I enjoyed the child selector section. It can be a bit challenging remembering descendant selectors, adjacent selectors, how to use :first-child, :last-child, +, >, :nth-of-type(), etc. The diagrams and images used really helped in understanding how these all work. I especially liked learning how to use the title attribute to grab and style that element on the page. I feel like it's easier and one less thing to create a class or id for.
